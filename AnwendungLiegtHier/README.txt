1. Entpacken Sie die .zip. Darin liegt die Anwendung.
2. Fügen Sie die Meldung*.csv Dateien in den selben Ordner wie die Anwendung
3. Führen Sie die Anwendung aus und folgen Sie den Anweisungen auf der Konsole. 
Hinweis: Es kann sein, dass das Anti Virus Programm die Datei beim Ausführen in die Quarantäne gibt. Um das zu verhindern sollten Sie eine Ausnahme für den gesamten Ordner erstellen.
4. Das Output file sollte generiert worden sein.
